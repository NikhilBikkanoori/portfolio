# How to Run This Project

This project is a standard static front-end website built with HTML, CSS, and JavaScript.

## Option 1: Open in Browser Directly (Easiest)
1. Navigate to the project folder (`d:\Nikhil Bikkanoori\portfolio2`) in your File Explorer.
2. Double-click the `index.html` file.
3. It will open and run in your default web browser (like Chrome, Edge, or Firefox).

## Option 2: Using VS Code Live Server (Recommended for Development)
1. Open the project folder in **Visual Studio Code**.
2. Go to the Extensions view (`Ctrl+Shift+X`) and install the **Live Server** extension (by Ritwick Dey).
3. Open `index.html` in the editor.
4. Right-click anywhere in the code and select **Open with Live Server**, or click the **Go Live** button at the bottom right of the VS Code window.
5. This will start a local server and automatically reload the page when you save changes to your files.

## Option 3: Using Python Local Server
If you have Python installed, you can run a simple server:
1. Open your terminal or command prompt.
2. Navigate to the project directory: `cd "d:\Nikhil Bikkanoori\portfolio2"`
3. Run the following command:
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and go to `http://localhost:8000`.
